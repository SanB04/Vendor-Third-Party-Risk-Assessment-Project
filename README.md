# Vendor Risk Management Framework & Third-Party Risk Assessment

## Project Overview
This project demonstrates the implementation of a simulated enterprise Vendor Risk Management (VRM) program for FinSure Technologies, a fintech organization operating in a regulated environment. The project focuses on identifying, assessing, monitoring, and managing risks associated with third-party vendors through structured governance and risk management practices.

## Project Deliverables
- Vendor Risk Management Policy
- Vendor Inventory
- Vendor Risk Register
- Risk Matrix
- Vendor Tier Classification

## Key Areas Covered
- Vendor onboarding and due diligence
- Third-party risk assessment
- Risk scoring and residual risk analysis
- Vendor tiering and classification
- Vendor lifecycle management
- Compliance and security oversight

## Tools & Concepts Used
- Microsoft Excel
- Microsoft Word
- Third-Party Risk Management (TPRM)
- Risk Assessment Methodologies
- Vendor Governance Frameworks

## Disclaimer
This project was developed for educational and portfolio purposes within a simulated enterprise environment.
